# PythonLibraries


## For Pandas

Adım Adım İşlem Listesi

1️⃣ Veri İnceleme (Exploration)

head() → İlk 5 satırı gör

info() → Veri tipleri ve eksik değerler

describe() → Sayısal özet

2️⃣ Veri Temizleme (Data Cleaning)

Eksik değerleri doldurma (fillna)

Gereksiz sütunları silme (drop)

Veri tiplerini düzeltme (astype)

3️⃣ Filtreleme ve Seçim (Filtering / Selecting)

Belirli koşullara göre satır seçme

İstediğin sütunları seçme

4️⃣ Gruplama ve Özetleme (Grouping / Aggregation)

groupby ile ortalama, toplam veya diğer özetleri hesaplama

5️⃣ Yeni Kolonlar Eklemek (Feature Engineering)

Yeni sütun oluşturma

Sayısal veya kategorik bilgileri gruplayarak yeni özellikler ekleme

6️⃣ Sıralama (Sorting)

sort_values ile en büyükten en küçüğe veya tam tersi sıralama

7️⃣ Basit Görselleştirme (Visualization)

Histogram (hist)

Bar grafiği (barplot)

Scatter plot veya Seaborn grafikleri
